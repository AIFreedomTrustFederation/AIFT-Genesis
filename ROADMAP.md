# Roadmap

## Phase 1: Genesis Pack

Create the canonical repository structure, doctrine, templates, manifests, schemas, examples, and generator specifications.

## Phase 2: Genesis Generator

Build commands that instantiate local-first trust workspaces from Genesis templates.

Example commands:

```text
genesis trust init
genesis business init
genesis community init
genesis repository init
```

## Phase 3: Mobox Integration

Make Mobox the personal workspace that can initialize, edit, and operate a local trust seed.

## Phase 4: Forge Integration

Make AIFT Forge the repository, build, package, issue, release, and federation engine for trust workspaces.

## Phase 5: Living Atlas

Map trusts, projects, missions, repositories, businesses, and communities around each user's Green Zone.

## Phase 6: Federation Protocols

Define safe, voluntary synchronization between local trusts and shared Federation layers.
