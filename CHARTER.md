# AIFT Genesis Charter

## Repository Purpose

AIFT Genesis is the canonical source repository for the AI Freedom Trust Federation genome.

It defines the shared doctrine, templates, schemas, manifests, examples, and generator specifications inherited by future Federation systems.

## Scope

This repository may contain:

- constitutional doctrine;
- philosophical foundations;
- scientific and topological design models;
- civilization and governance models;
- AI stewardship rules;
- trust seed templates;
- JSON manifests and schemas;
- working example trusts;
- generator and validation specifications.

## Non-Scope

This repository does not itself prove empirical claims, provide regulated professional advice, or operate production financial, legal, medical, custody, investment, securities, banking, or insurance services.

## Stewardship Rule

Genesis should remain simple enough to copy, inspect, fork, and adapt.

## Dependency Rule

Other Federation repositories may inherit Genesis patterns, but each repository should state what it has adopted, modified, or rejected.
