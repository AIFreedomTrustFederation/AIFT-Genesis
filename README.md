# AIFT Genesis

The source from which every trust, every AI agent, every application, and every future repository in the AI Freedom Trust Federation is generated.

AIFT Genesis is the constitutional and technical genome of the Federation. It defines the shared doctrine, templates, schemas, manifests, examples, and generator specifications inherited by Mobox, Forge, DynastyLink, BookSmith, the Living Atlas, and future trust instances.

## Core Principle

A user does not merely create an account. A user instantiates a local-first trust seed: a sovereign, inspectable copy of the Federation pattern scaled to one person, family, business, ministry, cooperative, community, repository, or project.

The whole Federation pattern exists in every local trust seed as structure, not as centralized ownership of private data.
